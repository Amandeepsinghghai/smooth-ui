### Basic

A basic switch, it works exactly like a `Radio`.

```js
<Switch />
```

### With labels

Add labels using `labeled` props.

```js
<Switch labeled />
```

### Disabled

Disable using `disabled` props.

```js
<FormCheck>
  <Switch id="switchExample1" disabled />
  <FormCheckLabel htmlFor="switchExample1">Switch it</FormCheckLabel>
</FormCheck>
```
