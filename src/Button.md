### Sizes

Set sizes using `size` prop like `"sm"` or `"lg"`.

```js
<div>
  <div style={{ margin: '5px' }}>
    <Button size="sm">Small</Button>
  </div>
  <div style={{ margin: '5px' }}>
    <Button>Medium</Button>
  </div>
  <div style={{ margin: '5px' }}>
    <Button size="lg">Large</Button>
  </div>
</div>
```

### Disabled

Disable using `disabled` prop.

```js
<Button disabled>Disabled</Button>
```
